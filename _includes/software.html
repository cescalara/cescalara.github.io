Software

I'm a big fan of open-source code and reproducible research. My code can be found on GitHub, and I list a few recent projects I've been working on below.

nu_coincidence:
A python package for investigating coincident source-neutrino detections with Monte Carlo simulations.
icecube_tools:
A python package for interacting with the publicly available data from the IceCube neutrino telescope, as well as running simulations and simple analyses.
QCDNUM.jl:
A julia wrapper for the QCDNUM software for fast QCD evolution and convolution.
PartonDensty.jl:
A julia package to extract parton distribution functions from accelerator data using Bayesian modeling.
